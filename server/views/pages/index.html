Before you start

This application is not meant as a complete framework, there will inevitably be some modification required.

Front end is built using GOV UK's Design System and their provided Nunjucks mixins. This can easily be swapped for something else.

Out of the box, the application assumes one form response per user, this may well be unsuitable to your needs.
